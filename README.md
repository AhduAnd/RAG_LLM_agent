# RAG обёртка для ИИ агента
Это проект по созданию RAG обёртки для YandexGPT, с помощью которой LLM сможет отвечать узконаправленные на вопросы, посвященные тематике определённого сайта. В качесте примера был выбран сайт базы знаний для продавцов Lamoda. Проект включает в себя реализацию парсера полезных статей с сайта, sparse+dense ретривера поверх собранной информации и минимально работающего ReAct цикла LLM агента на основе этого ретривера. Вся реализация живёт локально внутри .ipynb ноутбука, который был написан в Google Colab.
